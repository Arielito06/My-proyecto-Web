Paso 1: El usuario escribe una dirección en el navegador

¿Qué ocurre al ingresar una URL?

¿Sabías qué? El proceso completo de cargar una página web suele realizarse en menos de un segundo.

Explicación

Cuando ingresas una URL en el navegador, este primero consulta un servidor DNS para traducir el nombre de la página como "google.com" a una dirección IP numérica, que es la ubicación real del servidor donde está alojada la web. Luego, el navegador se conecta con ese servidor y le solicita la página mediante un mensaje HTTP. El servidor responde enviando el código HTML, CSS, JavaScript e imágenes, y el navegador interpreta ese código para organizar los elementos, aplicar los estilos, ejecutar las interacciones y finalmente mostrar la página completa en tu pantalla. Todo este proceso ocurre en fracciones de segundo.

Partes de una URL

Protocolo

Es la regla de comunicación entre el navegador y el servidor.

Dominio

Es el nombre único que identifica un sitio web.

Ruta

Indica la ubicación específica dentro del sitio web, como una carpeta o archivo.

Parámetros

Son datos adicionales que se envían al servidor.
