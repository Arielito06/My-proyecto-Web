Paso 2: El navegador pide la página al servidor

Solicitud HTTP

GET POST PUT DELETE

Métodos GET y POST

Método Descripción Cuándo se usa
GET Solicita datos sin modificar el servidor. Los parámetros van en la URL. Cargar páginas, imágenes, búsquedas, enlaces
POST Envía datos en el cuerpo de la petición. No se ven en la URL. Formularios de login, registro, comentarios, subir archivos

Ejemplo de solicitud GET

GET /index.html HTTP/1.1
Host: www.ejemplo.com
User-Agent: Chrome/120.0
Accept: text/html

Definición

HTTP define varios métodos de petición que indican la acción deseada sobre un recurso. Además de GET y POST, existen otros como PUT, DELETE, HEAD, CONNECT, OPTIONS, TRACE y PATCH. Cada uno tiene una semántica diferente: algunos son seguros (no modifican datos), otros son idempotentes donde varias solicitudes iguales producen el mismo resultado.
